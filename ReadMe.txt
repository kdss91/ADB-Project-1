The program consists of a Main class in Main.java file which takes two input relations bag1 and bag2 which are located in directory data.
The program implements TPMMS algorithm and performs sort-based bag difference.
Please put your set of input relations in files bag1 and bag2 located in folder data.
The program generates sorted sublists, complete sorted relations, and bagDifference in temp directory.
Please remove old files in the temp directory before running the program.
